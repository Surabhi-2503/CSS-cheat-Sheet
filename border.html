PROPERTY DESCRIPTION
border-width The border-width property sets the width of an element's four borders
border-style The border-style property sets the style of an element's four borders
border-color The border-color property sets the color of an element's four borders.
border-radius The border-radius property defines the radius of the element's corners.

CSS Syntax


border: border-width border-style border-color|initial|inherit;

BORDER-WIDTH

The border-width property sets the width of an element's four borders. This property can have from one to four values.

Always declare the border-style property before the border-width property.


SYNTAX

border-width: medium|thin|thick|length|initial|inherit;



BORDER-STYLE

The border-style property sets the style of an element's four borders.This property can have from one to four values.


SYNTAX

border-style: none|hidden|dotted|dashed|solid|double|grooveridge;



BORDER-COLOR

The border-color property sets the color of an element's four borders

Always declare the border-style property before the border-color property.


SYNTAX

border-color: color|transparent|initial|inherit;



BORDER-RADIUS

The border-radius property defines the radius of the element's corners.

This property can have from one to four values.


SYNTAX

border-radius: 1-4 length|% / 1-4 length|%|initial|inherit;
